Welcome to the Manim Documentation

Manim is a tool for creating precise animations programmatically. It is used to create educational math videos.

Getting Started

To get started with Manim, you will need to install it and set up a project. Follow these steps:

  1. Install Python and the required dependencies
  2. Clone the Manim repository
  3. Create a new project directory and copy the example files
  4. Run the example script to create your first animation

For more detailed instructions, see the Installation Guide.

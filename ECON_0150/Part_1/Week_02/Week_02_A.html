Variable Types

Categorical variable types

Binary: two categories (yes/no; true/false)

  • Employment Status (Employed, Unemployed)

Nominal: no inherent order

  • Employment Sector (Agriculture, Services, Unemployed).

Ordinal: meaningful order without consistent interval.

  • Economic Optimism (Very Pessimistic to Very Optimistic).

Variable Types

Numerical/Quantitative variable types

Discrete: countable numbers with meaningful intervals

  • Number of Children in a Household.

Continuous: quantities measurable on the reals.

  • Household Income in USD.

Example: Household Employment

Analyze the distribution of a binary categorical variable.

Dataset: Household Employment Survey

Key Variables:

  • Household ID: Unique identifier for each household.
  • Employment Status: Binary variable indicating employment status (Employed, Unemployed).

Frequency Counts

Employment Status:
Employed: 65
Unemployed: 35
  • Most Common Status: Employed
  • Least Common Status: Unemployed

Proportions

  • Proportional Breakdown:
    • Employed: 65%
    • Unemployed: 35%
  • Insights:
    • Most households have an employed member.
    • Unemployment rate is 35% in this dataset.

Visualization

  • Bar Chart: Employment Status Distribution
# Code for Bar Chart (Python)
import matplotlib.pyplot as plt

# Data for the chart
statuses = ["Employed", "Unemployed"]
counts = [65, 35]

# Create the bar chart
plt.bar(statuses, counts, color="skyblue")
plt.title("Employment Status Distribution")
plt.xlabel("Employment Status")
plt.ylabel("Count")
plt.show()

Summary

  • Key Takeaways:
    • Employed households dominate the dataset (65%).
    • Unemployment is relatively high at 35%.

Introduction

  • Dataset: Economic Optimism Survey of 100 Individuals
  • Purpose: Understand perceptions of economic conditions using an ordinal categorical variable.
  • Key Variables:
    • ID: Unique identifier for each individual.
    • Economic Optimism: Likert-scale rankings ranging from “Very Pessimistic” to “Very Optimistic.”

Dataset Overview

  • Data Summary:
    • Number of observations: 100
    • Categories (Ordered):
      • Very Pessimistic
      • Somewhat Pessimistic
      • Neutral
      • Somewhat Optimistic
      • Very Optimistic
Sample Data:
| ID   | Economic Optimism    |
|------|----------------------|
| B001 | Very Optimistic      |
| B002 | Neutral              |
| B003 | Somewhat Pessimistic |
| B004 | Somewhat Optimistic  |

Frequency Counts

Very Pessimistic: 10
Somewhat Pessimistic: 20
Neutral: 30
Somewhat Optimistic: 25
Very Optimistic: 15
  • Most Common Response: Neutral
  • Least Common Response: Very Pessimistic

Proportions

  • Proportional Breakdown:
    • Very Pessimistic: 10%
    • Somewhat Pessimistic: 20%
    • Neutral: 30%
    • Somewhat Optimistic: 25%
    • Very Optimistic: 15%
  • Insights:
    • Most respondents feel Neutral or Somewhat Optimistic.
    • Very Pessimistic responses are rare.

Visualization

  • Bar Chart: Economic Optimism Distribution
# Code for Bar Chart (Python)
import matplotlib.pyplot as plt

# Data for the chart
categories = ["Very Pessimistic", "Somewhat Pessimistic", "Neutral", "Somewhat Optimistic", "Very Optimistic"]
counts = [10, 20, 30, 25, 15]

# Create the bar chart
plt.bar(categories, counts, color="skyblue")
plt.title("Economic Optimism Distribution")
plt.xlabel("Economic Optimism Level")
plt.ylabel("Count")
plt.xticks(rotation=45)
plt.show()

Summary

  • Key Takeaways:
    • Neutral responses dominate the dataset (30%).
    • Responses are skewed slightly toward optimism (25% Somewhat Optimistic, 15% Very Optimistic).
    • Very Pessimistic responses are the least frequent (10%).

    • Household ID: Unique identifier for each household.
    • Household Income (USD): Annual income of each household (in USD).

Dataset Overview

  • Data Summary:
    • Number of observations: 100
    • Variable of interest: Household Income (Continuous Numerical)
Sample Data:
| Household ID | Household Income (USD) |
|--------------|-------------------------|

Descriptive Statistics

Summary Statistics:
- Minimum: $30,200.15
- Maximum: $119,800.90
- Mean: $72,530.40
- Median: $73,000.00
- Standard Deviation: $20,350.25
  • Insights:
    • The mean and median are close, indicating a relatively symmetric distribution.
    • There is a wide range of incomes, with some households earning nearly $120,000.

Visualization

  • Histogram: Distribution of Household Income
# Code for Histogram (Python)
import matplotlib.pyplot as plt

# Data for the chart
income = [round(random.uniform(30000, 120000), 2) for _ in range(100)]  # Example data

# Create the histogram
plt.hist(income, bins=10, color="skyblue", edgecolor="black")
plt.title("Distribution of Household Income")
plt.xlabel("Income (USD)")
plt.ylabel("Frequency")
plt.show()

Box Plot

  • Box Plot: Identify Outliers and Spread
# Code for Box Plot (Python)
plt.boxplot(income, vert=False, patch_artist=True, boxprops=dict(facecolor="lightblue"))
plt.title("Household Income Distribution")
plt.xlabel("Income (USD)")
plt.show()

Summary

  • Key Takeaways:
    • Income distribution is fairly symmetric, centered around $72,530.
    • There are no extreme outliers, but there is significant variability.
